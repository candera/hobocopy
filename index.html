hobocopy by candera

A free, open source backup tool for Windows.

Hobocopy is a free, open-source backup tool for Windows. It can copy files that are locked, so you can do things like back up your Outlook .pst files without closing Outlook.

Questions? Problems?

Visit the hobocopy Google group, or just send an email.
